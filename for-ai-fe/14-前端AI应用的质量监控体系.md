## 问题

如何建立一套针对前端 AI 应用的质量监控体系？

## 回答

### 一、AI 应用监控的特殊性

传统前端监控关注页面加载和用户交互，而 AI 应用还需要关注：

1. **流式传输**：SSE/WebSocket 连接的稳定性
2. **生成质量**：模型输出是否符合预期
3. **成本追踪**：Token 消耗与费用
4. **用户体验**：等待时间、中断行为

### 二、监控指标体系

#### 1. 性能指标

| 指标                           | 定义                                | 目标值         | 采集方式   |
| ------------------------------ | ----------------------------------- | -------------- | ---------- |
| **TTFT** (Time To First Token) | 从发送请求到收到第一个 Token 的时间 | < 500ms        | 时间戳差值 |
| **TPS** (Tokens Per Second)    | 每秒生成的 Token 数量               | > 30 TPS       | 计数器     |
| **Total Generation Time**      | 完整响应生成时间                    | < 10s (短回复) | 时间戳差值 |
| **Stream Interruption Rate**   | 流式传输中断率                      | < 1%           | 错误计数   |

```typescript
// metrics/aiPerformance.ts

interface AIPerformanceMetrics {
  ttft: number // 首字时间（毫秒）
  totalTime: number // 总生成时间
  tokenCount: number // 生成的 Token 数
  tps: number // 每秒 Token 数
  interrupted: boolean // 是否中断
  errorType?: string // 错误类型
}

class AIPerformanceTracker {
  private startTime: number = 0
  private firstTokenTime: number = 0
  private tokenCount: number = 0
  private conversationId: string

  constructor(conversationId: string) {
    this.conversationId = conversationId
  }

  /** 请求开始 */
  start(): void {
    this.startTime = performance.now()
    this.tokenCount = 0
  }

  /** 收到第一个 Token */
  onFirstToken(): void {
    this.firstTokenTime = performance.now()
  }

  /** 收到 Token */
  onToken(token: string): void {
    if (this.tokenCount === 0) {
      this.onFirstToken()
    }
    // 简单估算：平均每 4 个字符为 1 个 Token
    this.tokenCount += Math.ceil(token.length / 4)
  }

  /** 生成完成 */
  complete(interrupted: boolean = false, error?: string): AIPerformanceMetrics {
    const endTime = performance.now()
    const totalTime = endTime - this.startTime
    const ttft = this.firstTokenTime - this.startTime

    // 计算 TPS（排除 TTFT 时间）
    const generationTime = (endTime - this.firstTokenTime) / 1000
    const tps = generationTime > 0 ? this.tokenCount / generationTime : 0

    const metrics: AIPerformanceMetrics = {
      ttft,
      totalTime,
      tokenCount: this.tokenCount,
      tps,
      interrupted,
      errorType: error,
    }

    // 上报指标
    this.report(metrics)

    return metrics
  }

  private report(metrics: AIPerformanceMetrics): void {
    // 发送到监控服务
    navigator.sendBeacon(
      '/api/metrics/ai-performance',
      JSON.stringify({
        ...metrics,
        conversationId: this.conversationId,
        timestamp: Date.now(),
        userAgent: navigator.userAgent,
      }),
    )
  }
}
```

#### 2. 异常监控

```typescript
// monitors/aiErrorMonitor.ts

/** AI 应用特有的错误类型 */
enum AIErrorType {
  SSE_CONNECTION_FAILED = 'SSE_CONNECTION_FAILED',
  SSE_STREAM_INTERRUPTED = 'SSE_STREAM_INTERRUPTED',
  MODEL_API_ERROR = 'MODEL_API_ERROR',
  RATE_LIMIT_EXCEEDED = 'RATE_LIMIT_EXCEEDED',
  CONTEXT_LENGTH_EXCEEDED = 'CONTEXT_LENGTH_EXCEEDED',
  MARKDOWN_PARSE_ERROR = 'MARKDOWN_PARSE_ERROR',
  TOKEN_ESTIMATION_ERROR = 'TOKEN_ESTIMATION_ERROR',
  RESPONSE_VALIDATION_ERROR = 'RESPONSE_VALIDATION_ERROR',
}

interface AIErrorEvent {
  type: AIErrorType
  message: string
  stack?: string
  context: {
    conversationId?: string
    messageId?: string
    model?: string
    promptLength?: number
    responseLength?: number
  }
  timestamp: number
}

class AIErrorMonitor {
  private errorQueue: AIErrorEvent[] = []
  private flushInterval: number

  constructor() {
    // 批量上报，减少请求次数
    this.flushInterval = window.setInterval(() => this.flush(), 5000)
  }

  /** 记录 SSE 连接错误 */
  captureSSEError(error: Event, conversationId: string): void {
    this.capture({
      type: AIErrorType.SSE_CONNECTION_FAILED,
      message: 'SSE connection failed',
      context: { conversationId },
      timestamp: Date.now(),
    })
  }

  /** 记录流中断 */
  captureStreamInterruption(
    reason: string,
    conversationId: string,
    tokensReceived: number,
  ): void {
    this.capture({
      type: AIErrorType.SSE_STREAM_INTERRUPTED,
      message: `Stream interrupted: ${reason}`,
      context: {
        conversationId,
        responseLength: tokensReceived,
      },
      timestamp: Date.now(),
    })
  }

  /** 记录模型 API 错误 */
  captureModelError(statusCode: number, errorBody: any, model: string): void {
    let errorType = AIErrorType.MODEL_API_ERROR

    if (statusCode === 429) {
      errorType = AIErrorType.RATE_LIMIT_EXCEEDED
    } else if (errorBody?.error?.code === 'context_length_exceeded') {
      errorType = AIErrorType.CONTEXT_LENGTH_EXCEEDED
    }

    this.capture({
      type: errorType,
      message: errorBody?.error?.message || `Model API error: ${statusCode}`,
      context: { model },
      timestamp: Date.now(),
    })
  }

  /** 记录 Markdown 解析错误 */
  captureMarkdownError(error: Error, content: string): void {
    this.capture({
      type: AIErrorType.MARKDOWN_PARSE_ERROR,
      message: error.message,
      stack: error.stack,
      context: { responseLength: content.length },
      timestamp: Date.now(),
    })
  }

  private capture(event: AIErrorEvent): void {
    this.errorQueue.push(event)

    // 严重错误立即上报
    if (this.isCritical(event.type)) {
      this.flush()
    }
  }

  private isCritical(type: AIErrorType): boolean {
    return [AIErrorType.MODEL_API_ERROR, AIErrorType.RATE_LIMIT_EXCEEDED].includes(type)
  }

  private flush(): void {
    if (this.errorQueue.length === 0) return

    const events = [...this.errorQueue]
    this.errorQueue = []

    navigator.sendBeacon('/api/metrics/ai-errors', JSON.stringify(events))
  }

  destroy(): void {
    clearInterval(this.flushInterval)
    this.flush()
  }
}

// 全局单例
export const aiErrorMonitor = new AIErrorMonitor()
```

```typescript
// metrics/aiBusinessMetrics.ts

interface ConversationMetrics {
  conversationId: string
  turnCount: number // 对话轮数
  totalUserTokens: number // 用户输入 Token 总数
  totalAssistantTokens: number // 模型输出 Token 总数
  promptLengths: number[] // 每轮 Prompt 长度
  userFeedback: {
    thumbsUp: number
    thumbsDown: number
    regenerateCount: number // 重新生成次数
    copyCount: number // 复制次数
  }
  duration: number // 会话时长
}

class BusinessMetricsCollector {
  private conversations: Map<string, ConversationMetrics> = new Map()

  /** 开始新会话 */
  startConversation(conversationId: string): void {
    this.conversations.set(conversationId, {
      conversationId,
      turnCount: 0,
      totalUserTokens: 0,
      totalAssistantTokens: 0,
      promptLengths: [],
      userFeedback: {
        thumbsUp: 0,
        thumbsDown: 0,
        regenerateCount: 0,
        copyCount: 0,
      },
      duration: Date.now(),
    })
  }

  /** 记录一轮对话 */
  recordTurn(
    conversationId: string,
    promptLength: number,
    responseTokens: number,
  ): void {
    const metrics = this.conversations.get(conversationId)
    if (!metrics) return

    metrics.turnCount++
    metrics.promptLengths.push(promptLength)
    metrics.totalUserTokens += this.estimateTokens(promptLength)
    metrics.totalAssistantTokens += responseTokens
  }

  /** 记录用户反馈 */
  recordFeedback(
    conversationId: string,
    type: 'thumbsUp' | 'thumbsDown' | 'regenerate' | 'copy',
  ): void {
    const metrics = this.conversations.get(conversationId)
    if (!metrics) return

    switch (type) {
      case 'thumbsUp':
        metrics.userFeedback.thumbsUp++
        break
      case 'thumbsDown':
        metrics.userFeedback.thumbsDown++
        break
      case 'regenerate':
        metrics.userFeedback.regenerateCount++
        break
      case 'copy':
        metrics.userFeedback.copyCount++
        break
    }

    // 立即上报反馈事件
    this.reportFeedback(conversationId, type)
  }

  /** 结束会话 */
  endConversation(conversationId: string): void {
    const metrics = this.conversations.get(conversationId)
    if (!metrics) return

    metrics.duration = Date.now() - metrics.duration

    // 上报完整会话指标
    this.reportConversation(metrics)
    this.conversations.delete(conversationId)
  }

  private estimateTokens(charCount: number): number {
    // 中文约 1.5 字符/Token，英文约 4 字符/Token
    return Math.ceil(charCount / 2.5)
  }

  private reportFeedback(conversationId: string, type: string): void {
    fetch('/api/metrics/feedback', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        conversationId,
        type,
        timestamp: Date.now(),
      }),
    })
  }

  private reportConversation(metrics: ConversationMetrics): void {
    navigator.sendBeacon('/api/metrics/conversation', JSON.stringify(metrics))
  }
}

export const businessMetrics = new BusinessMetricsCollector()
```

### 三、链路追踪

```typescript
// tracing/aiTracer.ts

interface TraceContext {
  traceId: string
  spanId: string
  parentSpanId?: string
}

class AITracer {
  private traceId: string
  private spans: Map<string, SpanData> = new Map()

  constructor() {
    // 生成全局 TraceID
    this.traceId = this.generateId()
  }

  /** 创建新 Span */
  startSpan(name: string, parentSpanId?: string): SpanData {
    const span: SpanData = {
      spanId: this.generateId(),
      parentSpanId,
      name,
      startTime: Date.now(),
      attributes: {},
      events: [],
    }

    this.spans.set(span.spanId, span)
    return span
  }

  /** 记录 Span 事件 */
  addEvent(spanId: string, name: string, attributes?: Record<string, any>): void {
    const span = this.spans.get(spanId)
    if (!span) return

    span.events.push({
      name,
      timestamp: Date.now(),
      attributes,
    })
  }

  /** 结束 Span */
  endSpan(spanId: string, status: 'ok' | 'error' = 'ok'): void {
    const span = this.spans.get(spanId)
    if (!span) return

    span.endTime = Date.now()
    span.status = status

    // 上报到追踪服务
    this.reportSpan(span)
  }

  /** 获取追踪 Header，传递给后端 */
  getTraceHeaders(): Record<string, string> {
    return {
      'X-Trace-ID': this.traceId,
      'X-Request-ID': this.generateId(),
    }
  }

  /** 包装 SSE 请求，注入追踪信息 */
  traceSSERequest(
    url: string,
    options: RequestInit = {},
  ): {
    request: Request
    span: SpanData
  } {
    const span = this.startSpan('sse-request')

    const headers = new Headers(options.headers)
    headers.set('X-Trace-ID', this.traceId)
    headers.set('X-Span-ID', span.spanId)

    span.attributes = {
      url,
      method: options.method || 'GET',
    }

    return {
      request: new Request(url, { ...options, headers }),
      span,
    }
  }

  private generateId(): string {
    return `${Date.now().toString(36)}-${Math.random().toString(36).substr(2, 9)}`
  }

  private reportSpan(span: SpanData): void {
    fetch('/api/tracing/spans', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        traceId: this.traceId,
        ...span,
      }),
    }).catch(console.error)
  }
}

interface SpanData {
  spanId: string
  parentSpanId?: string
  name: string
  startTime: number
  endTime?: number
  status?: 'ok' | 'error'
  attributes: Record<string, any>
  events: Array<{
    name: string
    timestamp: number
    attributes?: Record<string, any>
  }>
}

// 使用示例
export function createTracedSSEConnection(
  url: string,
  body: any,
): {
  eventSource: EventSource | null
  cleanup: () => void
} {
  const tracer = new AITracer()
  const { request, span } = tracer.traceSSERequest(url, {
    method: 'POST',
    body: JSON.stringify(body),
  })

  tracer.addEvent(span.spanId, 'connection-start')

  // 使用 fetch + ReadableStream 替代 EventSource 以支持 POST
  let aborted = false

  fetch(request)
    .then(async (response) => {
      tracer.addEvent(span.spanId, 'connection-established', {
        status: response.status,
      })

      const reader = response.body?.getReader()
      if (!reader) return

      while (!aborted) {
        const { done, value } = await reader.read()
        if (done) break

        // 处理数据...
        tracer.addEvent(span.spanId, 'chunk-received', {
          size: value.length,
        })
      }

      tracer.endSpan(span.spanId, 'ok')
    })
    .catch((error) => {
      tracer.addEvent(span.spanId, 'error', { message: error.message })
      tracer.endSpan(span.spanId, 'error')
    })

  return {
    eventSource: null,
    cleanup: () => {
      aborted = true
    },
  }
}
```

### 四、监控数据采集优化

#### 1. 采样策略

```typescript
// sampling/sampler.ts

interface SamplingConfig {
  // 基础采样率（0-1）
  baseRate: number
  // 错误事件总是采样
  alwaysSampleErrors: boolean
  // 慢请求阈值（毫秒）
  slowThreshold: number
  // 慢请求采样率
  slowSampleRate: number
}

class AdaptiveSampler {
  private config: SamplingConfig
  private recentErrorRate: number = 0

  constructor(config: SamplingConfig) {
    this.config = config
  }

  /** 决定是否采样 */
  shouldSample(event: { isError: boolean; duration?: number; type: string }): boolean {
    // 错误总是采样
    if (event.isError && this.config.alwaysSampleErrors) {
      return true
    }

    // 慢请求提高采样率
    if (event.duration && event.duration > this.config.slowThreshold) {
      return Math.random() < this.config.slowSampleRate
    }

    // 根据当前错误率动态调整采样率
    const adjustedRate = this.getAdjustedRate()
    return Math.random() < adjustedRate
  }

  /** 动态调整采样率 */
  private getAdjustedRate(): number {
    // 错误率高时提高采样率
    if (this.recentErrorRate > 0.1) {
      return Math.min(1, this.config.baseRate * 2)
    }
    return this.config.baseRate
  }

  updateErrorRate(rate: number): void {
    this.recentErrorRate = rate
  }
}

// 全局采样器
export const sampler = new AdaptiveSampler({
  baseRate: 0.1, // 10% 基础采样
  alwaysSampleErrors: true,
  slowThreshold: 3000, // 3 秒
  slowSampleRate: 0.5, // 慢请求 50% 采样
})
```

#### 2. 数据聚合与批量上报

```typescript
// aggregation/metricsAggregator.ts

interface AggregatedMetrics {
  period: {
    start: number
    end: number
  }
  ttft: {
    p50: number
    p90: number
    p99: number
    avg: number
  }
  tps: {
    avg: number
    min: number
    max: number
  }
  errorRate: number
  requestCount: number
}

class MetricsAggregator {
  private buffer: AIPerformanceMetrics[] = []
  private aggregationInterval: number
  private maxBufferSize: number = 1000

  constructor(intervalMs: number = 60000) {
    this.aggregationInterval = intervalMs

    // 定期聚合上报
    setInterval(() => this.aggregate(), this.aggregationInterval)
  }

  add(metrics: AIPerformanceMetrics): void {
    // 采样过滤
    if (
      !sampler.shouldSample({
        isError: !!metrics.errorType,
        duration: metrics.totalTime,
        type: 'performance',
      })
    ) {
      return
    }

    this.buffer.push(metrics)

    // 防止内存溢出
    if (this.buffer.length > this.maxBufferSize) {
      this.aggregate()
    }
  }

  private aggregate(): void {
    if (this.buffer.length === 0) return

    const metrics = [...this.buffer]
    this.buffer = []

    // 计算聚合指标
    const aggregated: AggregatedMetrics = {
      period: {
        start: Math.min(...metrics.map((m) => m.timestamp || 0)),
        end: Math.max(...metrics.map((m) => m.timestamp || 0)),
      },
      ttft: this.calculatePercentiles(metrics.map((m) => m.ttft)),
      tps: {
        avg: this.avg(metrics.map((m) => m.tps)),
        min: Math.min(...metrics.map((m) => m.tps)),
        max: Math.max(...metrics.map((m) => m.tps)),
      },
      errorRate: metrics.filter((m) => m.errorType).length / metrics.length,
      requestCount: metrics.length,
    }

    this.report(aggregated)
  }

  private calculatePercentiles(values: number[]): {
    p50: number
    p90: number
    p99: number
    avg: number
  } {
    const sorted = values.sort((a, b) => a - b)
    return {
      p50: sorted[Math.floor(sorted.length * 0.5)],
      p90: sorted[Math.floor(sorted.length * 0.9)],
      p99: sorted[Math.floor(sorted.length * 0.99)],
      avg: this.avg(values),
    }
  }

  private avg(values: number[]): number {
    return values.reduce((a, b) => a + b, 0) / values.length
  }

  private report(aggregated: AggregatedMetrics): void {
    navigator.sendBeacon('/api/metrics/aggregated', JSON.stringify(aggregated))
  }
}

export const metricsAggregator = new MetricsAggregator()
```

### 五、React 监控 Hook 封装

```tsx
// hooks/useAIMonitoring.ts
import { useEffect, useRef, useCallback } from 'react'

interface UseAIMonitoringOptions {
  conversationId: string
  onMetrics?: (metrics: AIPerformanceMetrics) => void
}

export function useAIMonitoring({ conversationId, onMetrics }: UseAIMonitoringOptions) {
  const trackerRef = useRef<AIPerformanceTracker | null>(null)
  const abortControllerRef = useRef<AbortController | null>(null)

  // 初始化追踪器
  useEffect(() => {
    trackerRef.current = new AIPerformanceTracker(conversationId)
    businessMetrics.startConversation(conversationId)

    return () => {
      businessMetrics.endConversation(conversationId)
    }
  }, [conversationId])

  // 开始请求
  const startRequest = useCallback(() => {
    trackerRef.current?.start()
    abortControllerRef.current = new AbortController()

    return abortControllerRef.current.signal
  }, [])

  // 记录 Token
  const onToken = useCallback((token: string) => {
    trackerRef.current?.onToken(token)
  }, [])

  // 完成请求
  const completeRequest = useCallback(
    (error?: string) => {
      const metrics = trackerRef.current?.complete(false, error)
      if (metrics) {
        metricsAggregator.add(metrics)
        onMetrics?.(metrics)
      }
    },
    [onMetrics],
  )

  // 中断请求
  const abortRequest = useCallback(() => {
    abortControllerRef.current?.abort()
    const metrics = trackerRef.current?.complete(true, 'user_abort')
    if (metrics) {
      aiErrorMonitor.captureStreamInterruption(
        'user_abort',
        conversationId,
        metrics.tokenCount,
      )
    }
  }, [conversationId])

  // 记录用户反馈
  const recordFeedback = useCallback(
    (type: 'thumbsUp' | 'thumbsDown' | 'regenerate' | 'copy') => {
      businessMetrics.recordFeedback(conversationId, type)
    },
    [conversationId],
  )

  return {
    startRequest,
    onToken,
    completeRequest,
    abortRequest,
    recordFeedback,
  }
}

// 使用示例
function ChatComponent() {
  const { startRequest, onToken, completeRequest, recordFeedback } = useAIMonitoring({
    conversationId: 'conv-123',
  })

  const sendMessage = async (message: string) => {
    const signal = startRequest()

    try {
      const response = await fetch('/api/chat', {
        method: 'POST',
        body: JSON.stringify({ message }),
        signal,
      })

      const reader = response.body?.getReader()
      if (!reader) return

      while (true) {
        const { done, value } = await reader.read()
        if (done) break

        const text = new TextDecoder().decode(value)
        onToken(text)
      }

      completeRequest()
    } catch (error) {
      completeRequest(error.message)
    }
  }

  return (
    <div>
      {/* 聊天 UI */}
      <button onClick={() => recordFeedback('thumbsUp')}>👍</button>
      <button onClick={() => recordFeedback('thumbsDown')}>👎</button>
    </div>
  )
}
```

### 六、总结

| 监控维度 | 关键指标           | 采集方式    | 告警阈值     |
| -------- | ------------------ | ----------- | ------------ |
| **性能** | TTFT, TPS, 总时间  | 时间戳计算  | TTFT > 2s    |
| **异常** | SSE 中断率, 5xx 率 | 错误捕获    | > 1%         |
| **业务** | 点赞率, 重生成率   | 事件追踪    | 重生成 > 20% |
| **链路** | TraceID 关联       | Header 透传 | 排查工具     |

核心原则：

1. **采样控制**：高频数据必须采样，避免性能损耗
2. **批量上报**：聚合后上报，减少网络请求
3. **错误优先**：错误事件不采样，立即上报
4. **链路贯通**：前后端 TraceID 统一，方便排查
